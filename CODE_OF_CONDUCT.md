# Code of Conduct

## Our Standard

This project wants contributions, discussion, and review to stay:

* respectful
* technically honest
* constructive
* focused on the work

People should be able to ask basic questions, disagree about design, and point out problems without being mocked or personally diminished.

## Expected Behavior

Examples of behavior that help this project:

* assuming good faith while still being clear about problems
* giving specific technical feedback instead of vague dismissal
* keeping criticism focused on code, docs, behavior, and decisions
* making room for different levels of experience
* correcting mistakes without turning them into status contests

## Unacceptable Behavior

Examples of unacceptable behavior include:

* harassment, intimidation, or personal attacks
* insults, slurs, or demeaning language
* deliberately hostile or bad-faith review behavior
* repeated derailment of technical discussion
* publishing private information without permission
* sexualized language or attention in project spaces

## Scope

This code of conduct applies to project spaces, including:

* GitHub issues
* pull requests
* discussions
* review comments
* other public or private communication channels used for project collaboration

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that do not follow this code of conduct.

Responses may include:

* a warning
* removal of content
* temporary restriction from project interaction
* permanent removal from project spaces in severe or repeated cases

## Reporting

If you experience or witness unacceptable behavior, contact:

* [team@rightbusiness.com.sg](mailto:team@rightbusiness.com.sg)

Please include enough detail to help maintainers understand what happened.

## Maintainer Responsibility

Maintainers are expected to enforce this code of conduct fairly, consistently, and with attention to context.

The goal is not performative moderation.
The goal is to keep collaboration safe, serious, and productive.
